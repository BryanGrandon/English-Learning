# Nouns

A **noun** is a word that names **a person, animal, place, thing, event, or idea**. Nouns tell us **who** or **what** we are talking about.

<ContentGrid>
<Callout type='example'>

- **Person:** teacher
- **Animal:** dog
- **Place:** school
- **Thing:** table
- **Event:** party
- **Idea:** happiness

</Callout>

<Callout type="example_sentences">

- The **teacher** is in the classroom.
- My **dog** is sleeping.
- We live in a **house**.
- The **party** starts at 7 p.m.
- **Happiness** is important.

</Callout>
</ContentGrid>

## Singular and Plural Nouns

A noun can be **singular** (one) or **plural** (more than one).

| Singular | Plural  |
| -------- | ------- |
| car      | cars    |
| house    | houses  |
| pen      | pens    |
| camera   | cameras |

Most plural nouns are formed by adding -s at the end of the word.

---

## Plural Rules

### Add -s

Most nouns form the plural by adding **-s**.

<Grammar type="rule" title="General Rule" highlight='Noun + s' >

- #### Examples
  - car → cars
  - house → houses
  - pen → pens
  - book → books

</Grammar>

---

### Consonant + y → ies

If a noun ends in **consonant + y**, change **y** to **ies**.

<Grammar type="rule" title="General Rule" highlight='consonant + y → ies' >

- #### Examples
  - party → parties
  - city → cities
  - baby → babies
  - country → countries

</Grammar>

<Callout type="tip"> 
If a noun ends in a **vowel + y**, simply add **-s**.

- boy → boys
- key → keys
- day → days
- toy → toys

</Callout>

---

### Add -es

If a noun ends in **s, ss, sh, ch, x, or o**, add **-es**.

<Grammar type="rule" title="General Rule" highlight='noun + es' >

- #### Examples
  - bus → buses
  - box → boxes
  - watch → watches
  - dish → dishes
  - tomato → tomatoes

</Grammar>

### f / fe → ves

Some nouns ending in **f** or **fe** change to **ves** in the plural.

<Grammar type="rule" title="General Rule" highlight='f / fe → ves' >

- #### Examples
  - leaf → leaves
  - wife → wives
  - life → lives
  - wolf → wolves

</Grammar>

---

## Irregular Plural Nouns

Some nouns have **irregular plural forms** and do not follow a rule.

| Singular | Plural   |
| -------- | -------- |
| man      | men      |
| woman    | women    |
| child    | children |
| person   | people   |
| tooth    | teeth    |
| foot     | feet     |
| mouse    | mice     |
