# Time

Time is used to express _hours and minutes_ during the day.

- #### We use time to:
  - tell the current time
  - ask and answer what time it is
  - talk about daily routines
  - describe schedules and appointments

<Callout type="example_sentences">

- It's three o'clock.
- What time is it?
- School starts at eight.
- I go to bed at ten.

</Callout>

<Callout type="remember">

Learning how to tell the time helps you talk about **daily activities**, **school**, **work**, and **appointments**.

</Callout>

---

## Time Vocabulary

Before learning how to tell the time, know these common words.

| Word    | Meaning                        |
| ------- | ------------------------------ |
| hour    | 60 minutes                     |
| minute  | One of the 60 parts of an hour |
| clock   | A device that shows time       |
| o'clock | Used for exact hours           |
| past    | Minutes after the hour         |
| to      | Minutes before the next hour   |
| quarter | 15 minutes                     |
| half    | 30 minutes                     |
| a.m.    | Before noon                    |
| p.m.    | After noon                     |

## Reading Time (Digital)

The most common way to tell the time is to say: **Hour + Minutes**

| Time | Read             |
| ---: | ---------------- |
| 2:10 | two ten          |
| 4:25 | four twenty-five |
| 5:01 | five oh one      |
| 7:05 | seven oh five    |
| 7:40 | seven forty      |
| 9:09 | nine oh nine     |
| 9:55 | nine fifty-five  |

<Callout type="note">

When the minutes are between **01** and **09**, English speakers often say **oh** before the minute.

- #### Examples:
  - 5:01 → five **oh** one
  - 7:05 → seven **oh** five
  - 9:09 → nine **oh** nine

</Callout>

## Exact Hours (o'clock)

Use **o'clock** only when there are _no minutes_.

<LessonGrid className='md:grid-cols-3'>
<Callout type="example">

| Time  | Read         |
| :---: | ------------ |
| 1:00  | one o'clock  |
| 5:00  | five o'clock |
| 10:00 | ten o'clock  |

</Callout>
<div className='col-span-2'>
  <Grammar
    type="commonMistake"
    title="Don't use *o'clock* with minutes"  
  >

| ❌ Incorrect              | ✅ Correct        |
| ------------------------- | ----------------- |
| six o'clock fifteen       | six fifteen       |
| quarter past four o'clock | quarter past four |
| ten to three o'clock      | ten to three      |

  </Grammar>
</div>
</LessonGrid>

## Traditional Time

English also has a traditional way to tell the time.

<LessonGrid className='sm:grid-cols-2'>
  <TopicBlock icon={ArrowRightLeft} title="Past"  >

      Use **past** for 1-30 _minutes after the hour_.

      | Time | Read              |
      | :---: | ----------------- |
      | 2:05 | five past two     |
      | 3:10 | ten past three    |
      | 7:20 | twenty past seven |

  </TopicBlock>
  <TopicBlock icon={ArrowRightLeft} title="To"  >

      Use **to** for 31-59 _minutes before the next hour_.

      |  Time | Read            |
      | :----: | --------------- |
      |  4:50 | ten to five     |
      |  8:45 | fifteen to nine |
      | 11:55 | five to twelve  |

  </TopicBlock>
</LessonGrid>

## Special Expressions

Some times have **special expressions** instead of saying the exact minutes.

These expressions are very common in everyday English.

| Time | Expression         |
| ---: | ------------------ |
| 3:15 | quarter past three |
| 6:30 | half past six      |
| 7:45 | quarter to eight   |

<LessonGrid className="sm:grid-cols-2">

<Callout type="tip" title="Remember">

- **quarter past** = 15 minutes **after** the hour
- **half past** = 30 minutes **after** the hour
- **quarter to** = 15 minutes **before** the next hour

</Callout>

</LessonGrid>

## Time of Day

Use **a.m.** and **p.m.** to show whether a time is before or after noon.

| Abbreviation | Meaning                                   |
| ------------ | ----------------------------------------- |
| a.m.         | Midnight (12:00 a.m.) → Noon (12:00 p.m.) |
| p.m.         | Noon (12:00 p.m.) → Midnight (12:00 a.m.) |

### Parts of the Day

English also uses different expressions for each part of the day.

<LessonGrid className='md:grid-cols-2'>

| Part      | Approximate Time       |
| --------- | ---------------------- |
| Morning   | 6:00 a.m. – 12:00 p.m. |
| Afternoon | 12:00 p.m. – 6:00 p.m. |
| Evening   | 6:00 p.m. – 9:00 p.m.  |
| Night     | 9:00 p.m. – 6:00 a.m.  |

<Callout type="example">

- 7:00 a.m. → seven in the morning
- 1:00 p.m. → one in the afternoon
- 6:30 p.m. → six thirty in the evening
- 10:00 p.m. → ten at night

</Callout>

</LessonGrid>

## Remember

<Callout type="remember">

- Read the **hour first**, then the **minutes**.
- Use **o'clock** only for exact hours.
- Use **past** after the hour.
- Use **to** before the next hour.
- Learn both digital and traditional expressions.
- Practice by reading clocks every day.

</Callout>
