# Quantifiers

**_Quantifiers_** are words that tell us _how many_ or _how much_ of something there is.

They usually _come before the noun_ they modify.

<Callout type="example_sentences">

- There are **some apples** in the fridge.
- There is **much water** in the bottle.

</Callout>

---

## Quantifiers Overview

| Quantifier         | Countable | Uncountable |
| ------------------ | :-------: | :---------: |
| many               |    ✅     |     ❌      |
| much               |    ❌     |     ✅      |
| some               |    ✅     |     ✅      |
| any                |    ✅     |     ✅      |
| no / none          |    ✅     |     ✅      |
| a lot of / lots of |    ✅     |     ✅      |
| few / a few        |    ✅     |     ❌      |
| little / a little  |    ❌     |     ✅      |

<Callout type="tip">

- **_Countable nouns_** → things you can count (_books, apples, cars_).
- **_Uncountable nouns_** → things you cannot count individually (_water, milk, money_).

</Callout>

Some quantifiers are used only with **_countable nouns_**, others only with **_uncountable nouns_**, and some can be used with **both**.

---

## Quantifiers Table

| Quantifier             | Main Use                                    |
| ---------------------- | ------------------------------------------- |
| **many**               | Questions and negative sentences            |
| **much**               | Questions and negative sentences            |
| **some**               | Affirmative sentences, offers, and requests |
| **any**                | Questions and negative sentences            |
| **no / none**          | Express zero quantity                       |
| **a lot of / lots of** | Informal alternative to _many_ and _much_   |
| **few / a few**        | Express a small number                      |
| **little / a little**  | Express a small amount                      |

---

## Common Quantifiers

### Many

- Used with plural countable nouns.
- Common in questions and negatives.

### Much

- Used with uncountable nouns.
- Common in questions and negatives.

### Some

- Used in affirmative sentences.
- Also used in offers and requests.

---

## Common Mistakes

<Grammar type="common_mistake" title='Using much with countable nouns'>

- ❌ much books
- ✅ **many books**

</Grammar>

### Confusing few and little

- few → **_countable nouns_**
- little → **_uncountable nouns_**

<Callout type="example_sentences">

- few students.
- little time.

</Callout>
