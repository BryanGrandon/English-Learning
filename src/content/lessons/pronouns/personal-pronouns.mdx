# Personal Pronouns

Los pronombres personales son palabras breves que se utilizan para sustituir a los sustantivos que se refieren a personas, lugares o cosas,
con el fin de evitar repeticiones. Varían según la persona, el número, el género y el caso gramatical (sujeto u objeto).

## 1. Subject Pronouns

Los pronombres de sujeto son pronombres personales que _realizan la acción_ en una oración.

<section className='grid grid-cols-1 gap-4 sm:grid-cols-2'>
  <LessonBlock variant='position'>
    - Antes del verbo
  </LessonBlock>

  <LessonBlock variant='structure'>    
  - *Subject* + verb + Complement      
  </LessonBlock>
</section>

| Word | Meaning           | Example                     |
| ---- | ----------------- | --------------------------- |
| I    | yo                | **I** am ill.               |
| you  | tú, usted         | **You** are tall.           |
| he   | él                | **He** is handsome.         |
| she  | ella              | **She** is pretty.          |
| it   | ello (neutral)    | **It** is cold today.       |
| we   | nosotros          | **We** are tired.           |
| you  | vosotros, ustedes | **You** are angry.          |
| they | ellos, ellas      | **They** are at the cinema. |

## 2. Object Pronouns

Los pronombres de objeto sustituyen al objeto de una oración, es decir, a la persona o cosa que recibe la acción del verbo.

<section className='grid grid-cols-1 gap-4 sm:grid-cols-2'>
  <LessonBlock variant='position'>
      - Vienen después del verbo.
      - Después de preposiciones como: **for**, **to**, **with**, **in**.
  </LessonBlock>

  <LessonBlock variant='structure'>    
      - *Subject* + verb + *Object Pronoun*
      - Preposition + *Object Pronoun*
  </LessonBlock>
</section>

| Word | Meaning               | Example               |
| ---- | --------------------- | --------------------- |
| me   | mi                    | Can you help **me**?  |
| you  | a ti, a usted         | I can help **you**.   |
| him  | a él                  | Can you see **him**?  |
| her  | a ella                | Give it to **her**.   |
| it   | a ello                | Give **it** a kick.   |
| us   | a nosotros            | Can you see **us**?   |
| you  | a vosotros, a ustedes | I see **you**.        |
| them | a ellos               | He can help **them**. |

## 3. Neutral Form

En inglés, los pronombres distinguen el género:

- **He** → male
- **She** → female
- **It** → neutral (things, animals, or situations)

La _forma neutra_ se utiliza para referirse a cosas, ideas, situaciones, el tiempo, la hora, la distancia, o cuando se desconoce el género o este no es relevante.

<section className='grid grid-cols-1 gap-4 lg:grid-cols-2'>

| Use of it | Examples                  |
| --------- | ------------------------- |
| Weather.  | It is raining.            |
| Time.     | What time is it?          |
| Distance. | It is far.                |
| Things.   | Where is it _[the book]_? |

  <LessonBlock variant='important'>

      - En inglés suele ser necesario un pronombre de sujeto.
      <div className='ml-4'>
        - ❌ **_Is raining._**
        - ✅ **_It is raining._**
      </div>

      - Usa **it** para cosas en singular y **they** para cosas en plural.
      <div className='ml-4'>
        - The book is new. **It** is interesting.
        - The books are new. **They** are interesting.
      </div>

  </LessonBlock>
</section>

## 4. Subject vs Object Pronouns

Los pronombres de sujeto y de objeto pueden referirse a la misma persona o cosa, pero cumplen funciones diferentes dentro de la oración.

- El _subject pronoun_ realiza la acción.
- El _object pronoun_ recibe la acción.

### Ejemplo

- **She** called **me**.
  - **She** → subject pronoun (realiza la acción)
  - **Me** → object pronoun (recibe la acción)

| Subject | Object |
| ------- | ------ |
| I       | me     |
| you     | you    |
| he      | him    |
| she     | her    |
| it      | it     |
| we      | us     |
| they    | them   |

<LessonBlock variant='important'>

- Utiliza un **_subject pronoun_** antes del verbo.
- Utiliza un **_object pronoun_** después del verbo o de una preposición.

<p className='py-2'>Ejemplos:</p>

<div className='ml-4'>

- ✅ **He** likes her.
- ✅ She spoke to **him**.
- ❌ Him likes her.

</div>

</LessonBlock>
