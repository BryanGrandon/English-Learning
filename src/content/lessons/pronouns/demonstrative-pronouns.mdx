# Demonstrative Pronouns

Los pronombres demostrativos se utilizan para indicar o señalar algo, como si estuvieras señalándolo con el dedo.

## 1. Demonstrative Pronouns

Los pronombres demostrativos nos ayudan a señalar a personas, cosas o ideas concretas. Indican si algo está **cerca** o **lejos**, y si es **singular** o **plural**.

| Word  | Meaning               | Type     | Distance |
| ----- | --------------------- | -------- | -------- |
| this  | este/a/o              | singular | near     |
| that  | ese/a/o / aquel/lla   | singular | far      |
| these | estos/as              | plural   | near     |
| those | esos/as / aquellos/as | plural   | far      |

### How They Work

<section className='grid grid-cols-1 gap-4 sm:grid-cols-2'>
<GrammarPoint title='Near + Singular → **this**' >
Usa **this** para algo que tengas a mano.

- This car is new. (Este coche es nuevo.)

</GrammarPoint>

<GrammarPoint title='Far + Singular → **that**' >
Usa **that** para referirte a algo que está lejos de ti.

- That car is new. (Ese coche es nuevo.)

</GrammarPoint>

<GrammarPoint title='Near + Plural → **these**' >
Usa **these** para varias cosas que tengas a mano.

- These cars are new. (Estos coches son nuevos.)

</GrammarPoint>

<GrammarPoint title='Far + Plural → **those**' >
Usa **those** para referirte a varias cosas que están lejos de ti.

- Those cars are new. (Aquellos coches son nuevos.)

</GrammarPoint>

</section>

## 2. Use Demonstrative Pronouns with or without a noun

Los pronombres demostrativos (**this, that, these, those**) pueden desempeñar dos funciones:

### With a Noun (Determiner Use)

En este caso, actúan como _determinantes demostrativos_ porque van _antes de un sustantivo_ y lo modifican.

<LessonColumns className='sm:flex-row'>
  <LessonBlock variant='structure'>Demonstrative + noun</LessonBlock>
  
  <LessonBlock variant='example'>
    
  - **This car** is new. → Este coche es nuevo.
  - **Those shoes** are expensive. → Esos zapatos son caros.

  </LessonBlock>
</LessonColumns>

### Without a Noun (Pronoun Use)

En este caso, actúan como _pronombres demostrativos_ porque sustituyen al sustantivo.

<LessonColumns className='sm:flex-row'>
  <LessonBlock variant='structure'>Demonstrative + verb</LessonBlock>

  <LessonBlock variant='example'>

    - **This** is a good book. → Este es un buen libro.
    - What is that? → Que es eso?

  </LessonBlock>
</LessonColumns>

## 3. Other Common Uses of Demonstratives

<section className='grid grid-cols-1 gap-4 sm:grid-cols-2'>
<GrammarPoint title='Introducing yourself on the phone' >
Usa **this** para decir quién eres.

- Hello, **this is** Alicia.
  - ❌ Don’t say: _I am Alicia_

</GrammarPoint>

<GrammarPoint title='Asking who someone is' >
Usa **that** cuando la persona no esté cerca o no se vea bien.

- Peter, **is that you?**
  - 📌 Used when there is distance or uncertainty.

</GrammarPoint>

<GrammarPoint title='Introducing people' >
Usa **this** para presentar a alguien que esté a tu lado.

- Lucy, **this is** my friend Jill.
  - 📌 The person is next to you.

</GrammarPoint>

<GrammarPoint title='Talking about the past' >
Usa **that** para referirte a algo que ya ha terminado o que ocurrió hace tiempo.

- **That** pizza was delicious. 📌 It can refer to:
  - Something you just experienced
  - Something further in the past

</GrammarPoint>

</section>
