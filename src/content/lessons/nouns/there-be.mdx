## What is **there be**

he structure **there + be** is used to say that **something exists**.

In Spanish it usually means **“hay”**.

📌 _Examples._

- **There is** a cat.
- **There are** four chairs.

In English the verb **be changes depending on singular or plural nouns**.

## There is

**There is** is used with:

- **singular countable nouns**
- **uncountable nouns**

There is also a short form: there’s

📌 _Countable nouns._

- There is a pencil.
- There’s one car.
- There is not an apple.
- Is there a pen?

📌 _Uncountable nouns._

- There is milk.
- There is not time.
- Is there sugar?

## There are

**There are** is used only with **plural countable nouns**.

📌 _Examples._

- There are five pencils.
- There are two cars.
- There are not many people.
- Are there many students?

⚠️ There is **no short form for “there are”**.

### Sentence Structure

**_Affirmative_**

🏗️ _Structure._

- There + be + noun

📌 _Examples._

- There is a book on the table.
- There are three cats in the house.

**_Negative_**

🏗️ _Structure._

- There + be + not + noun

📌 _Examples._

- There is not a problem.
- There are not any apples.

**_Interrogative_**

🏗️ _Structure._

- Be + there + noun ?

📌 _Examples._

- Is there a bank near here?
- Are there any restaurants?

---

## There vs It

Use **there** when we mention something **for the first time** or when we say that something exists.

Use **it** when we talk about **something already mentioned**.

📌 _Examples._

- There is a pen on the table. **It** is my pen.
- There is a car in the garage. **It** is blue.
- There is milk. **It** is in the refrigerator.

---

## Common Mistakes

**_Using there is with plural nouns_**

- ❌ There is two cars
- ✅ **There are two cars**

**_Using there are with singular nouns_**

- ❌ There are a cat
- ✅ **There is a cat**

**_Confusing there and it_**

- ❌ It is a cat on the table
- ✅ **There is a cat on the table**
