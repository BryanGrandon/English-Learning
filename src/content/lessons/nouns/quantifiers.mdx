## What are Quantifiers?

**Quantifiers** are words used to express the **quantity of a noun**. They answer the question **“How many?” or “How much?”**.

Like articles, quantifiers **come before the noun** they modify.

📌 _Examples._

- There are **some apples** in the fridge.
- There is **much water** in the bottle.

---

## Quantifiers Overview

| Quantifier         | Countable | Uncountable |
| ------------------ | --------- | ----------- |
| many               | ✅        |             |
| much               |           | ✅          |
| some               | ✅        | ✅          |
| any                | ✅        | ✅          |
| no / none          | ✅        | ✅          |
| a lot of / lots of | ✅        | ✅          |
| little / a little  |           | ✅          |
| few / a few        | ✅        |             |

Some quantifiers are used only with **countable nouns**, others with **uncountable nouns**, and some with **both**.

---

## Quantifiers Table

| Quantifier            | Use                      | Example                       |
| --------------------- | ------------------------ | ----------------------------- |
| **many**              | plural countable (− / ?) | There aren’t **many people**. |
| **much**              | uncountable (− / ?)      | There isn’t **much time**.    |
| **some**              | affirmative              | I have **some money**.        |
| **any**               | negative / questions     | Do you have **any sugar**?    |
| **no / none**         | zero quantity            | There are **no books**.       |
| **a lot of**          | all nouns                | She has **a lot of friends**. |
| **little / a little** | uncountable              | We have **little time**.      |
| **few / a few**       | countable                | There are **few cars**.       |

---

## Common Mistakes

**_Using much with countable nouns_**

- ❌ much books
- ✅ **many books**

**_Using many with uncountable nouns_**

- ❌ many money
- ✅ **much money**

**_Confusing few and little_**

- **few → countable nouns**
- **little → uncountable nouns**

📌 _Examples._

- few students
- little time
